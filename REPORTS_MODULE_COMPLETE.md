## 🔧 How to Use

### Viewing Reports Online

1. **Navigate to Reports**
   - Click "Reports" in sidebar
   - Or go to admin/reports.php

2. **Select Report Type**
   - Choose from dropdown:
     - Sales Report
     - Top Medicines
     - Purchase Report
     - Customer Report

3. **Set Date Range**
   - From Date: Start date
   - To Date: End date
   - Default: Current month

4. **Generate Report**
   - Click "Generate Report" button
   - Report displays instantly
   - Statistics cards update
   - Table shows data

### Downloading Excel Report

1. **Generate Report First**
   - Select report type
   - Set date range
   - Click "Generate Report"

2. **Click Download Button**
   - Click "📥 Download Excel" button
   - File downloads automatically
   - Saves to Downloads folder

3. **Open in Excel**
   - Locate downloaded file
   - Filename: `{type}_report_{date}.xls`
   - Example: `sales_report_2024-03-15.xls`
   - Double-click to open in Excel

### Printing Reports

1. **Generate Report**
   - Select type and date range
   - Click "Generate Report"

2. **Click Print Button**
   - Click "🖨️ Print Report" button
   - Or press Ctrl+P
   - Print dialog opens

3. **Print Settings**
   - Select printer
   - Choose orientation (Landscape recommended)
   - Set margins
   - Click Print

## 📊 Database Queries

### Sales Report Query
```sql
SELECT DATE(sale_date) as date, 
       COUNT(*) as total_sales, 
       SUM(grand_total) as revenue, 
       SUM(tax_amount) as total_tax,
       SUM(discount_amount) as total_discount 
FROM sales 
WHERE DATE(sale_date) BETWEEN ? AND ? 
GROUP BY DATE(sale_date) 
ORDER BY date DESC
```

### Top Medicines Query
```sql
SELECT m.medicine_name, 
       SUM(si.quantity) as total_sold,
       SUM(si.subtotal) as total_revenue
FROM sale_items si 
JOIN medicines m ON si.medicine_id = m.id 
JOIN sales s ON si.sale_id = s.id 
WHERE DATE(s.sale_date) BETWEEN ? AND ?
GROUP BY si.medicine_id 
ORDER BY total_sold DESC 
LIMIT 20
```
