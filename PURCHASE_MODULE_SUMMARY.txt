================================================================================
                            HOW TO USE
================================================================================

1. ADD SUPPLIERS (If not already added)
   - Go to Suppliers menu
   - Add supplier details
   - Save supplier

2. CREATE A PURCHASE
   - Navigate to Purchases → New Purchase
   - Select supplier
   - Set purchase date
   - Add medicines from dropdown
   - Edit batch numbers if needed
   - Adjust unit prices if needed
   - Set quantities
   - Add tax percentage if applicable
   - Select payment status
   - Add notes (optional)
   - Click "Complete Purchase"

3. VIEW PURCHASES
   - Go to Purchases list
   - Browse all purchases
   - Click "View" for details

4. STOCK AUTOMATICALLY UPDATED
   - Medicine quantity increased
   - Purchase price updated
   - Batch number updated
   - Status set to available

================================================================================
                        PAYMENT STATUS OPTIONS
================================================================================

✅ PAID (Green Badge)
   - Full payment received
   - Use when payment completed

⚠️ PENDING (Yellow Badge)
   - No payment received yet
   - Use for credit purchases

⚠️ PARTIAL (Yellow Badge)
   - Some payment received
   - Use for advance payments

================================================================================
                          CALCULATIONS
================================================================================

Total Amount = Sum of (Unit Price × Quantity) for all items
Tax Amount = (Total Amount × Tax %) / 100
Grand Total = Total Amount + Tax Amount

Example:
- Item 1: 100 units @ Rs 20 = Rs 2,000
- Item 2: 50 units @ Rs 80 = Rs 4,000
- Total Amount: Rs 6,000
- Tax (5%): Rs 300
- Grand Total: Rs 6,300
