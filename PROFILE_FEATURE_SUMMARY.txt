================================================================================
                              HOW TO USE
================================================================================

UPDATE PROFILE:
1. Go to profile page
2. Edit full name, email, or phone
3. Click "💾 Update Profile"
4. Success message appears
5. Changes saved immediately

CHANGE USERNAME:
1. Scroll to "Change Username" section
2. Enter new username
3. Enter current password
4. Click "🔄 Change Username"
5. System validates uniqueness
6. Success or error message

CHANGE PASSWORD:
1. Scroll to "Change Password" section
2. Enter current password
3. Enter new password (min 6 chars)
4. Confirm new password
5. Watch real-time match indicator:
   - ✅ Green = Match
   - ❌ Red = No match
6. Click "🔒 Change Password"
7. Success or error message
